Para productos elaborados, primero debe crear una receta

Los precios de venta para productos elaborados se vinculan directamente a la receta. Esto permite calcular automáticamente los costos y gestionar los precios de manera eficiente.

  2. Configurar Ingredientes: Definir los ingredientes de la receta
  3. Crear Precios de Venta: Una vez con receta, podrá gestionar precios
